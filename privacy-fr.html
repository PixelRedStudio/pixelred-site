Politique de confidentialité – TapSafe

Dernière mise à jour : janvier 2025

Pixel Red Studio (« nous », « notre ») exploite l’application mobile TapSafe. Cette politique de confidentialité décrit quelles données sont collectées, comment elles sont utilisées et quels sont vos droits.

1. Responsable du traitement

Le responsable du traitement des données est Pixel Red Studio. Pour toute question relative à cette politique ou à vos données personnelles, vous pouvez nous contacter à contact@pixelredstudio.com ou via le site pixelredstudio.com.

2. Données collectées et finalités

2.1 Données stockées uniquement sur votre appareil

Les données suivantes sont enregistrées localement sur votre téléphone (fichier de sauvegarde de l’application) et ne sont pas envoyées à nos serveurs :

  • Scores et records (modes Classique, Contre-la-montre, Survie)
  • Préférences (musique, effets sonores, vibrations, langue)
  • Progression (niveau, XP, pièces, objets achetés, missions quotidiennes, récompenses quotidiennes)
  • Paramètres d’affichage (skins, thèmes)

Ces données servent au bon fonctionnement du jeu et à votre progression. Vous pouvez les supprimer en désinstallant l’application ou en effaçant les données de l’app.

2.2 Publicité (Google AdMob)

TapSafe affiche des publicités gérées par Google AdMob. AdMob peut collecter ou utiliser notamment :

  • Identifiant publicitaire (IDFA/AAID)
  • Données sur l’appareil et l’utilisation de l’application (pour la diffusion et la mesure des annonces)

Ces données sont traitées par Google selon sa Politique de confidentialité Google. Nous utilisons les annonces pour financer le jeu ; vous pouvez gérer vos préférences de personnalisation des annonces (consentement, refus, gestion des options) via le message de consentement affiché dans l’application (réglementations européennes / RGPD).

2.3 Analytics (Firebase / Google)

Nous utilisons Firebase Analytics (Google) pour comprendre comment l’application est utilisée, sans vous identifier personnellement. Les données peuvent inclure :

  • Événements anonymisés (par exemple : ouverture de l’app, fin de partie, rétention jour 1 / jour 7)
  • Informations techniques (type d’appareil, version de l’application, langue)

Ces données nous aident à améliorer le jeu et l’expérience utilisateur. Le traitement est conforme à la politique Firebase / Google.

2.4 Classements et compte Google Play

Si vous utilisez les classements en ligne (Google Play Games), Google traite les données liées à votre compte Google et aux scores que vous soumettez. Ce traitement est régi par les conditions et la politique de confidentialité de Google.

3. Base légale (RGPD)

Pour les utilisateurs dans l’Espace économique européen :

  • Consentement : pour la publicité personnalisée et, le cas échéant, pour certains analytics, lorsque nous affichons un bandeau ou un message de consentement.
  • Exécution du contrat / intérêt légitime : pour les données strictement nécessaires au fonctionnement du jeu (sauvegarde locale, classements, progression).

4. Partage des données

Nous ne vendons pas vos données personnelles. Les données peuvent être transmises à :

  • Google (AdMob, Firebase, Play Games) pour la publicité, les analytics et les classements, conformément à leurs politiques.
  • Les autorités, si la loi l’exige.

5. Conservation des données

  • Données locales : tant que l’application est installée ; suppression possible en effaçant les données de l’app ou en désinstallant.
  • Données AdMob / Firebase / Google : selon les durées et politiques de Google (voir les liens indiqués ci-dessus).

6. Vos droits (RGPD et autres)

Selon votre pays, vous pouvez notamment :

  • Accéder aux données vous concernant
  • Demander la rectification ou la suppression
  • Vous opposer ou restreindre certains traitements
  • Retirer votre consentement (pour les traitements fondés sur le consentement)
  • Déposer une réclamation auprès d’une autorité de contrôle (en France : CNIL)

Pour la publicité : utilisez le message « Gérer les options » / « Manage options » dans l’app ou les paramètres de votre appareil (ex. « Réinitialiser l’identifiant publicitaire »). Pour nous contacter : contact@pixelredstudio.com.

7. Enfants

TapSafe ne cible pas intentionnellement les enfants en bas âge. Si vous êtes parent ou tuteur et que vous estimez que nous avons collecté des données relatives à un enfant sans consentement approprié, contactez-nous pour demander leur suppression.

8. Modifications

Nous pouvons mettre à jour cette politique de confidentialité. La date de dernière mise à jour figure en haut de la page. Les changements importants pourront être signalés dans l’application ou sur notre site.
